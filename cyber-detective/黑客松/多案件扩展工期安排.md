## 一、项目变更概述

### 变更前（已完成）
| 维度 | 内容 |
|------|------|
| 案件数 | 1 个（霓虹公寓谋杀案） |
| 场景 | 4 个（body / corridor / desk / workshop） |
| 嫌疑人 | 3 个（林小北 / 赵明月 / 钱一鸣） |
| 结局 | 3 种（好/普通/坏，由单案评分触发） |
| 主线 | 无 |

### 变更后
| 维度 | 内容 |
|------|------|
| 案件数 | 4 个（3 主线 + 1 隐藏） |
| 场景 | 案件1 = 4 场景（已有），案件2/3 = 各 3 场景，隐藏案件待定 |
| 嫌疑人 | 案件1 = 3 人（已有），案件2/3 = 各 3 人，隐藏案件待定 |
| 结局 | 每案件 3 种分支 + 最终整体结局（由全案结果累积决定） |
| 主线 | 侦探事务所 Hub + 新闻简报 + 赛博义眼 AI 助手 |
